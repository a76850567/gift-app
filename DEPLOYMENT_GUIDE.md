## ⭐ 方案 1：在线部署（最推荐）

### 为什么选择在线部署？
- ✅ **最专业**：有独立网址，可以分享链接
- ✅ **最方便**：任何设备都能访问（手机、平板、电脑）
- ✅ **最稳定**：不担心现场电脑出问题
- ✅ **最快速**：5分钟搞定，完全免费

---

### 🌐 Vercel 部署（推荐）

#### 步骤：

1. **访问 Vercel**
   - 打开 https://vercel.com
   - 点击右上角 "Sign Up"（注册）

2. **登录账号**
   - 可以用 GitHub、GitLab 或 Email 登录
   - 建议用 GitHub，以后更方便

3. **上传项目**
   - 点击 "Add New..." → "Project"
   - 选择 "Import Git Repository" 或 "Upload"
   - 如果没用 Git，选择右上角的 "Deploy"
   - 直接拖拽你的整个项目文件夹

4. **配置项目**
   - Framework Preset: 选择 "Vite"
   - Root Directory: ./
   - Build Command: `npm run build`
   - Output Directory: `dist`
   - 点击 "Deploy"

5. **等待部署**
   - 大约 1-2 分钟
   - 完成后会显示你的网址
   - 例如：`https://gift-app-xxxx.vercel.app`

6. **测试**
   - 点击链接查看应用
   - 用手机扫二维码测试移动端

#### 优点：
- 自动 HTTPS（安全）
- 全球 CDN 加速
- 自动更新（如果用 Git）
- 免费域名

---

### 🎨 Netlify 部署（备选）

#### 步骤：

1. **访问 Netlify**
   - 打开 https://www.netlify.com
   - 点击 "Sign Up"

2. **登录账号**
   - 可以用 GitHub、GitLab 或 Email

3. **部署项目**
   - 进入 Dashboard
   - 找到 "Sites" 页面
   - 直接**拖拽整个项目文件夹**到网页中间的区域

4. **等待部署**
   - 大约 2-3 分钟
   - 完成后会显示你的网址
   - 例如：`https://gift-app-xxxx.netlify.app`

#### 优点：
- 极其简单（拖拽上传）
- 自动 HTTPS
- 免费域名
- 支持自定义域名

## 💻 方案 2：本地运行

### 适合情况：
- 演示现场有稳定网络
- 演示电脑你可以提前配置
- 想展示 "开发者" 的专业感

### 前置要求：

1. **安装 Node.js**
   - 访问 https://nodejs.org/
   - 下载 "LTS" 版本（左边那个）
   - 安装时一路点 "Next"

2. **验证安装**
   - 打开命令行（Windows：Win+R 输入 cmd）
   - 输入：`node --version`
   - 应该显示版本号，如：`v20.x.x`

### 运行步骤：

#### Windows 用户（最简单）：
1. 解压项目文件夹
2. 双击 `START_APP.bat`
3. 等待浏览器自动打开
4. 完成！

#### Mac/Linux 用户：
1. 打开终端
2. 进入项目文件夹：`cd /path/to/gift-app`
3. 运行启动脚本：
   ```bash
   chmod +x START_APP.sh
   ./START_APP.sh
   ```
4. 等待浏览器自动打开

#### 手动运行（所有系统）：
```bash
# 1. 进入项目目录
cd /path/to/gift-app

# 2. 安装依赖（第一次运行）
npm install

# 3. 启动开发服务器
npm run dev

# 4. 在浏览器打开显示的地址
# 通常是：http://localhost:3000
```

### 注意事项：
- ⚠️ 第一次运行 `npm install` 需要 5-10 分钟
- ⚠️ 需要网络连接（下载依赖）
- ⚠️ 演示前一定要提前测试
- ⚠️ 演示时不要关闭命令行窗口

## 🆘 常见问题解决

### Q1: Vercel 部署失败怎么办？
A: 
- 检查所有文件是否都上传了
- 确认 `package.json` 文件存在
- 尝试 Netlify 作为备选

### Q2: 本地运行报错怎么办？
A:
```bash
# 清除并重新安装
rm -rf node_modules package-lock.json
npm install

# 或者使用 npm cache clean
npm cache clean --force
npm install
```

### Q3: 应用打开是空白页面？
A:
- 检查浏览器控制台（F12）
- 确认是否有 JavaScript 错误
- 尝试清除浏览器缓存（Ctrl+Shift+Delete）

### Q4: 手机访问应用样式不对？
A:
- 应用已经适配移动端
- 检查是否用了旧浏览器
- 建议用 Chrome/Safari
